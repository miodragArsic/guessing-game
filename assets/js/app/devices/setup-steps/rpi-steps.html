1. Prepare hardware & software Learn more

  1. Place the Grove shield on top of the RPI
  2. Attach Rotary knob to A0 pin on the shield
  3. Attach LED to D4 pin on the shield
  4. Set up RPI OS

2. Use FTP to install libraries Learn more

  1. Connect Raspberry Pi to internet using LAN cable
  2. Connect to Raspberry Pi using any FTP client. 

    Host: [your_rpi_ip_address]

    Protocol: SFTP

    User: pi

    Password: raspberry

  3. Choose and download library
  4. Extract zipped library
  5. Copy extracted files to Raspberry Pi using FTP client
Download

3. run First application script

This is your first_app. Download this script, include it in directory where you installed library and run the script.

Your device will be activated as soon as the first data arrives in the cloud.

Note: It's important to place First_app.py in same directory with ATT_IOT.py
