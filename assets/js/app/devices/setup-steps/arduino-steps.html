1. Prepare hardware & software Learn more

  1. Place the Grove shield on top of the Arduino
  2. Attach Rotary knob to A0 pin on the shield
  3. Attach LED to D4 pin on the shield
  4. Set up Arduino

2. Install libraries Learn more

  1. Download libraries (ATT_IOT, pubsubclient)
  2. Unzip libraries
  3. Open Arduino IDE
  4. Under arduino-client-master/library you will find 
    ATT_IOT
    and 
    pubsubclient
    folders
  5. Import both using 
    sketch > Include library > Add .ZIP library
Download

3. run First application script

This is your first_app. Download this script, open it in the Arduino IDE and run the script.

Your device will be activated as soon as the first data arrives in the cloud.

Note: In case you have multiple devices running, change the MAC-address in the application code to make sure they don't interfere with each other.
