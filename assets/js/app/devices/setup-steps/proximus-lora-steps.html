1.Claim your LoRa™ device Learn more

  1. Set up an account as Proximus Developer
  2. Claim your device with the DEVEUI code on the box

2. Prepare the environment Learn more

  1. Install the Arduino IDE and SODAQ Mbili

3. Download and install libraries Learn more

  1. Download the AllThingsTalk LoRa™ client library
  2. In the 
    ATT_LoRa_IOT/examples
    folder you will find
    1. basic demo.ino sketch in demo
    2. example sketch for every sensor in sensor tests
    3. sketches for 3 experiments in experiments
  3. Unzip libraries
  4. Open Arduino IDE
  5. Go to 
    sketch > Include library > Add .ZIP library
  6. browse to the arduino-LoRa-master folder, select it and import
  7. Repeat this process to add all needed libraries.
